出海第一步，搞定技术栈。

本仓库只收录独立开发者出海技术栈和工具，持续整理中……

**收录标准：**

- 帮助独立开发者提升开发效率
- 帮助独立开发者降低成本
- 市场上足够流行 

欢迎提 pr 和 issues 更新。



## Web 开发模板
| 模板 | 特性 |
| --- | --- |
| [smart-excel-ai（免费）](https://github.com/weijunext/smart-excel-ai) | Next.js。集成了登录、支付（lemon squeezy）、AI功能 |

| [Shipfast（付费）](https://shipfa.st/) | Next.js。集成了登录、支付（stripe）、邮件、AI功能 |
